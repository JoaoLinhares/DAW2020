ABSTRACT:

XML é um formato para a criação de documentos com dados organizados de forma hierárquica, frequentemente em documentos de texto formatos. Sendo uma linguagem bem formatada é possível através de um conjunto de procedimentos transformar esse mesmo ficheiro XML em um ficheiro HTML ao qual a nossa informação poderá estar bem apresentada para o utilizador através de browsers e de outros aplicativos que os suportem.

Para esta transformação são criados documentos XSL em que ao aplicar uma transformação com esse documento a um ficheiro XML é possível obter o conteúdo HTML correspondente.

Para ajudar na criação do ficheiro XSL, foi criado um documento DTD que valide o ficheiro XML que pretendemos transformar, para perceber a estrutura do respetivo e facilitar na construção do ficheiro XSL.

Deliverables:

  • Ficheiro XML
  • DTD que valida o ficheiro XML
  • Ficheiro XSLT
  • HTML gerado
  • Pasta com ficheiros CSS para o HTML gerado
